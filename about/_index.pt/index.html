Sobre Hugo

Hugo é um gerador de site escrito em Go.

Ele utiliza vários projetos open source, incluindo:

  • Cobra
  • Viper
  • J Walter Weatherman
  • Cast

Aprenda mais e contribua no GitHub.

Setup

Algumas curiosidades sobre Hugo:

  • Feito com Go
  • Loosely inspired by Jekyll
  • Inicialmente desenvolvido by spf13 em um trem enquanto viajava de e para Manhattan.
  • Codificado no Vim usando spf13-vim
